# منصة معمار الرقمية

تطبيق PWA متقدم (Mobile-First) يربط العملاء بالمهندس لإدارة الطلبات الهندسية مع نظام دردشة متكامل.

## المميزات

- ✅ دعم اللغة العربية والإنجليزية مع تبديل سلس
- ✅ نظام ثيمات (ليلي/نهاري)
- ✅ نظام مصادقة مبسط باستخدام localStorage
- ✅ ثلاث باقات (أساسية، متقدمة، احترافية)
- ✅ نموذج طلب ذكي متعدد المراحل (15 حقل)
- ✅ نظام دردشة متكامل مع رفع الملفات
- ✅ ضغط الملفات لتسهيل الرفع والتحميل
- ✅ لوحة تحكم للمهندس
- ✅ نظام حماية المخرجات (Security Gate)

## التقنيات المستخدمة

- **Frontend**: Next.js 14, React, TypeScript, Tailwind CSS
- **Storage**: localStorage (يمكن الترقية إلى قاعدة بيانات لاحقاً)
- **i18n**: next-intl
- **Forms**: react-hook-form, zod
- **Icons**: lucide-react

## الإعداد

1. تثبيت المتطلبات:
```bash
npm install
```

2. تشغيل المشروع:
```bash
npm run dev
```

الآن سيعمل المشروع على `http://localhost:3000`

**ملاحظة**: المشروع حالياً يستخدم localStorage لحفظ البيانات. لاستخدام قاعدة بيانات، يمكن الترقية لاحقاً.

## البنية

```
src/
├── app/                    # Next.js App Router
│   ├── [locale]/          # صفحات محلية مع دعم i18n
│   │   ├── (auth)/        # صفحات المصادقة
│   │   ├── (client)/      # صفحات العميل
│   │   └── (admin)/       # لوحة تحكم المهندس
│   └── layout.tsx         # Layout الرئيسي
├── components/
│   ├── ui/                # مكونات UI أساسية
│   ├── forms/             # مكونات النماذج
│   ├── chat/              # مكونات الدردشة
│   ├── admin/             # مكونات لوحة التحكم
│   ├── account/           # مكونات الحساب
│   ├── auth/              # مكونات المصادقة
│   ├── layout/            # مكونات التخطيط
│   └── security/          # نظام الحماية
├── lib/
│   ├── storage.ts         # وظائف التخزين (localStorage)
│   ├── fileCompression.ts # ضغط الملفات
│   └── utils.ts           # وظائف مساعدة
├── contexts/              # React Contexts (Auth, Theme)
├── types/                 # TypeScript types
├── i18n/                  # إعدادات i18n
└── locales/               # ملفات الترجمة (ar.json, en.json)
```

## النشر

1. بناء المشروع:
```bash
npm run build
```

2. تشغيل الإنتاج:
```bash
npm start
```

3. نشر على Vercel/Netlify:
المشروع جاهز للنشر مباشرة بدون الحاجة لإعدادات إضافية.

## ملاحظات

- البيانات حالياً محفوظة في localStorage (متصفح المستخدم)
- يمكن الترقية لقاعدة بيانات في المستقبل
- النظام يدعم اللغة العربية والإنجليزية بشكل كامل
